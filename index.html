Project Overview

The web application will provide the ability for the user to select several parameters which will help them select their next bottle, such as: region of interest, type of wine and price point / budget. The application will display the Top 10 rated wines in a bar chart, another chart with all the wines available and a map showing the concentration of wines by region/country of the world.

Develop a wine selection dashboard to provide wine enthusiasts have a better understanding of where many notable wines originate, the associated price points, the WineEnthusiast rating (100-point scale), and tasters description of the wine.
